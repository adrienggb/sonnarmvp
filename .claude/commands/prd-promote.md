Avance un PRD dans le cycle de vie : 01-draft → 02-ready → 03-in-progress → 04-complete.

Arguments : nom du fichier PRD (ex: PRD-02-portail-client) ou numéro (ex: 02).

Étapes :
1. Trouve le fichier PRD dans son dossier actuel
2. Vérifie que les dépendances du PRD sont bien dans le dossier amont requis
3. Met à jour le champ **Statut** dans le fichier
4. Déplace le fichier vers le dossier suivant
5. Met à jour le tableau statut dans prds/01-draft/BACKLOG.md

Si aucun argument, demande quel PRD promouvoir en listant ceux disponibles.
